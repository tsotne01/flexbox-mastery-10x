# ­Ъје CSS Flexbox Mastery - 10 Levels

рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ! рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ **CSS Flexbox**-рЃўрЃА рЃАрЃарЃБрЃџрЃДрЃЮрЃцрЃўрЃџрЃљрЃЊ рЃљрЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљрЃерЃў. рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ 10 рЃЊрЃЮрЃюрЃўрЃАрЃњрЃљрЃю (Level), рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃЊрЃљрЃю рЃарЃЌрЃБрЃџрЃўрЃАрЃЎрЃћрЃю.

## ­ЪЊѓ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

- `index.html` - рЃљрЃЦ рЃљрЃарЃўрЃА рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЏрЃ«рЃљрЃарЃћ (HTML). **рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃљрЃЏ рЃцрЃљрЃўрЃџрЃА рЃ«рЃћрЃџрЃў рЃљрЃа рЃљрЃ«рЃџрЃЮрЃЌ!** РЏћ
- `style.css` - рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃцрЃљрЃўрЃџрЃў. рЃЎрЃЮрЃЊрЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЦ. РюЇ№ИЈ

## ­Ъџђ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

1.  рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃЮрЃџрЃЊрЃћрЃарЃў **VS Code**-рЃерЃў.
2.  рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ `style.css` рЃцрЃљрЃўрЃџрЃў.
3.  рЃЕрЃљрЃЏрЃЮрЃДрЃћрЃЋрЃўрЃЌ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃА. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃџрЃћрЃЋрЃћрЃџрЃА рЃљрЃЦрЃЋрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: `/* CODE HERE */`.
4.  рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃЕрЃљрЃгрЃћрЃарЃЮрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў Flexbox рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў (properties), рЃарЃљрЃЌрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃЌ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ.
5.  рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў `index.html`-рЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃўрЃЌ.

---

## ­ЪЈє рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃў (Levels)

рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ рЃцрЃљрЃарЃљрЃЋрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЌрЃћрЃЏрЃљрЃА:

- [ ] **Level 1:** Flexbox-рЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃЊрЃљрЃерЃЮрЃарЃћрЃЉрЃћрЃЉрЃў
- [ ] **Level 2:** рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ
- [ ] **Level 3:** рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ
- [ ] **Level 4:** рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃў рЃфрЃћрЃюрЃбрЃарЃўрЃарЃћрЃЉрЃљ
- [ ] **Level 5:** рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃЋрЃћрЃбрЃљрЃЊ
- [ ] **Level 6:** рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ
- [ ] **Level 7:** рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ рЃљрЃ«рЃљрЃџ рЃ«рЃљрЃќрЃќрЃћ
- [ ] **Level 8:** рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃерЃћрЃЋрЃАрЃћрЃЉрЃљ
- [ ] **Level 9:** рЃўрЃюрЃЊрЃўрЃЋрЃўрЃЊрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ
- [ ] **Level 10:** рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃарЃўрЃњрЃўрЃЌрЃЮрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ

---

­ЪњА **рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ:** рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљрЃерЃў рЃњрЃљрЃўрЃГрЃћрЃЊрЃћрЃЌ, рЃњрЃљрЃўрЃ«рЃАрЃћрЃюрЃћрЃЌ `display: flex;` рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃќрЃћ рЃЊрЃљ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў рЃЕрЃљрЃљрЃбрЃљрЃарЃћрЃЌ `justify-content` рЃЊрЃљ `align-items` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃќрЃћ.

**рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃћрЃЉрЃў!** ­Ъњ╗
